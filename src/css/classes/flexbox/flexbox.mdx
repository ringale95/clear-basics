## Definition
 
The flexible box layout module (usually referred to as flexbox) is a ```one-dimensional layout model``` (row or col)for distributing space between items and includes numerous alignment capabilities. This article gives an outline of the main features of flexbox, which we will explore in more detail in the rest of these guides.

## Two axes of flexbox

- main
    - Defined by flex-direction with 4 possible values:
        - row
        - column
        - row-reverse
        - column-reverse

## Default property values

- flex direction : row
- flex-grow : 0
- flex-shrink : 1
- align-items: stretch
- flex-basis: auto
- flex-wrap: nowrap

## Properties

- <b>Flex grow</b>
    - Flex-grow CSS property sets flex grow factor, which specifies how much of flex container's free space if any should 
    be assigned to flex item's main size

    <br/>
    <h3> Example of flex-grow/shrink</h3>
        <b> yellow grow - 1: violet shrink - 1: cyan grow - 1 </b>

    <div class="container-grow">
        <div class="boxflex-item1 boxflex"> </div>
        <div class="boxflex-item2 boxflex"> </div>
        <div class="boxflex-item3 boxflex"> </div>
    </div>
    <br/>

- <b>Flex wrap</b>
     - The flex-wrap CSS property sets whether flex items are forced onto one line or can wrap onto multiple lines. If wrapping is allowed, it sets the direction that lines are stacked.

        - Values

            - nowrap
                - The flex items are laid out in a single line which may cause the flex container to overflow.
            - wrap
                - The flex items break into multiple lines
            - wrap-reverse
                - Behaves the same as wrap, but cross-start and cross-end are inverted.
